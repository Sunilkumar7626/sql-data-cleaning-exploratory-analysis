# SQL Layoffs Data Analysis

## Overview

This project analyzes a global layoffs dataset using MySQL to identify trends in workforce reductions across companies, industries, and time periods. The analysis includes data cleaning, aggregation, ranking, and trend analysis using SQL.

---

## Objectives

* Identify companies with the highest layoffs
* Analyze layoffs by industry and company stage
* Examine monthly and yearly layoff trends
* Find companies with 100% workforce layoffs
* Rank top companies by layoffs each year

---

## SQL Skills Demonstrated

* Data Cleaning
* Common Table Expressions (CTEs)
* Window Functions
* ROW_NUMBER()
* DENSE_RANK()
* Aggregate Functions (SUM, AVG, MIN, MAX)
* Date Functions
* Rolling Totals
* Trend Analysis

---

## Key Analyses

* Total layoffs by company
* Total layoffs by industry
* Layoffs by funding stage
* Monthly and yearly layoff trends
* Rolling cumulative layoffs over time
* Top 5 companies by layoffs for each year
* Companies with complete workforce reductions
