# SQL Data Cleaning Project — Layoffs Dataset

## Overview
This project focuses on cleaning a raw layoffs dataset using SQL in MySQL.  
The goal was to transform messy data into a clean, analysis-ready dataset for reporting and visualization.

---

## Tools Used
- SQL
- MySQL
- Microsoft Excel / CSV

---

## Key Tasks Performed
- Removed duplicate records using `ROW_NUMBER()`
- Standardized text values (company, industry, country)
- Converted date column into SQL `DATE` format
- Handled null and blank values
- Removed unnecessary rows and columns

---

## SQL Concepts Used
- Window Functions
- CTEs
- Joins
- String Functions
- Date Functions
- Data Cleaning Techniques
